# Color-of-words
> 2020년 1학기 경희대학교 포스트모던아트읽기 기말 프로젝트

웹 크롤러를 사용하여 해당 홈페이지 및 아티클 등을 읽어오고, 이를 분석해 자소에 따른 색을 부여합니다. 자소의 사용을 시각화하여 글의 톤을 분석합니다.
